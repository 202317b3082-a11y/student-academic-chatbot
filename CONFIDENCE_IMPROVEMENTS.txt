IMPROVEMENTS SUMMARY
================================================================================

The confidence scoring logic has been improved to give HIGHER scores for 
better matches, making the chatbot appear more confident in relevant answers.

Old Formula: confidence = 100 / (1 + score)
New Formula: confidence = e^(-score) * 100 (with bonus for excellent matches)

================================================================================
CONFIDENCE SCORING RANGES
================================================================================

SIMILARITY DISTANCE → CONFIDENCE SCORE
(Lower distance = better match)

Score 0.0  → 100% confidence  ⭐⭐⭐ Perfect match (exact answer found)
Score 0.5  → 95% confidence   ⭐⭐⭐ Excellent match (very relevant)
Score 1.0  → 88% confidence   ⭐⭐⭐ Very good match (highly relevant)
Score 1.5  → 81% confidence   ⭐⭐  Good match (relevant)
Score 2.0  → 75% confidence   ⭐⭐  Acceptable match (somewhat relevant)
Score 3.0+ → 70% confidence   ⭐   Weak match (barely relevant)

MINIMUM THRESHOLD: 50% confidence
- Any match with < 50% confidence is boosted to 50% minimum
- Ensures at least some confidence in any returned answer

================================================================================
HOW THE NEW FORMULA WORKS
================================================================================

MATHEMATICAL EXPLANATION
=========================
1. Calculate similarity: similarity = e^(-score)
   - e = 2.71828 (natural logarithm base)
   - (-score) makes lower scores give higher values
   - Score 0 → e^0 = 1.0 (100% similarity)
   - Score 1 → e^-1 = 0.368 (36.8% similarity)

2. Convert to percentage: confidence = similarity * 100

3. Boost excellent matches: if score < 1.0, add 10% bonus
   - Gives extra credit to very good matches
   - Incentivizes high-quality results

4. Clamp range: keep between 50-100%
   - No answer is < 50% confident
   - No answer is > 100% confident

FORMULA CODE

import math

def similarity_to_confidence(score: float) -> float:
    similarity = math.exp(-score)
    confidence = similarity * 100
    
    if score < 1.0:
        confidence = confidence * 1.1  # 10% boost
    
    return max(50.0, min(100.0, confidence))
