## Tips for Better Searches

1. **Use Exact Names**: Search for "IEEE" not "electrical engineering journal"
2. **Try Shorter Terms**: Search "exam" not "when is comprehensive examination"
3. **Search Topic + Action**: "Pearson + access" or "exam + makeup"
4. **Use Keywords**: "OpenAthens", "registration", "fee", "exam"
5. **Combine Terms**: "OpenAthens activation", "ERP login issue"

## Advanced Search Tips

1. **Wildcard Search**: "exam" finds "exam", "examinations", "make-up exam"
2. **Partial Matching**: "open" finds "OpenAthens"
3. **Case Insensitive**: "EBSCO", "ebsco", "Ebsco" all work
4. **Multiple Words**: "OpenAthens activation" searches both terms
5. **Empty Search**: Shows all policies when field is empty

## For Admins: Policy Management

### When to Update Policies
- When procedures change
- When new platforms are added
- When contact information updates
- When deadlines change
- When errors are reported

### When to Delete Policies
- When platforms are discontinued
- When procedures become obsolete
- When replaced by newer policies
- Only delete if absolutely necessary

### When to Add Policies
- When new e-resources are added
- When new procedures are introduced
- When frequently asked questions arise
- When policies are requested by students
