================================================================================
WHAT ARE EMBEDDINGS?
================================================================================

Embeddings are vector representations of text that enable:
- Fast semantic search
- Similarity matching
- Efficient indexing
- Better search results

When you update:
- College policies → Need to regenerate policy embeddings (create_embedding.py)
- Feedback data → Need to regenerate feedback embeddings (feedback_embedding.py)

USER WORKFLOW
================================================================================

ADMIN GENERATES POLICY EMBEDDINGS
==================================
1. Admin updates college policies via "Update Knowledge Base" section
2. After making changes, clicks "⚡ Generate Embeddings" button (purple)
3. Button shows "⏳ Generating..." and becomes disabled
4. Backend runs create_embedding.py in background
5. Embeddings regenerated for all policy updates
6. Success alert: "✅ Policy embeddings generated successfully!"
7. Button returns to normal state

ADMIN GENERATES FEEDBACK EMBEDDINGS
====================================
1. User feedback accumulates in feedback.csv
2. Admin goes to "Feedback View" section
3. After reviewing feedback, clicks "⚡ Generate Embeddings" button (orange)
4. Button shows "⏳ Generating..." and becomes disabled
5. Backend runs feedback_embedding.py in background
6. Embeddings regenerated for all feedback entries
7. Success alert: "✅ Feedback embeddings generated successfully!"
8. Button returns to normal state

TECHNICAL DETAILS
================================================================================

BACKEND IMPLEMENTATION
======================
Endpoints use subprocess.run() to execute Python scripts:

POST /admin/generate-embeddings/policy
  - Requires: JWT token + admin role
  - Executes: python create_embedding.py
  - Timeout: 300 seconds (5 minutes)
  - Output: Stdout/stderr captured and returned
  
POST /admin/generate-embeddings/feedback
  - Requires: JWT token + admin role
  - Executes: python feedback_embedding.py
  - Timeout: 300 seconds (5 minutes)
  - Output: Stdout/stderr captured and returned

Response format:
{
  "message": "Success message",
  "output": "Script output (stdout)"
}

On error:
{
  "message": "Error message",
  "error": "Error details (stderr)"
}

================================================================================
WHAT GETS GENERATED
================================================================================

POLICY EMBEDDINGS (from create_embedding.py)
=============================================
Input: college_policy.txt
Output: 
  - /faiss_index/index.faiss (FAISS vector database)
  - /faiss_index/index.pkl (metadata)
Purpose: Fast semantic search over policies

Process:
  1. Read college_policy.txt
  2. Split text into chunks (500 chars with 50 char overlap)
  3. Generate embeddings using sentence-transformers model
  4. Store in FAISS vector index
  5. Save to /faiss_index/ directory

FEEDBACK EMBEDDINGS (from feedback_embedding.py)
================================================
Input: feedback.csv
Output:
  - /faiss_feedback_index/index.faiss (FAISS vector database)
  - /faiss_feedback_index/index.pkl (metadata)
Purpose: Feedback analysis and pattern detection

Process:
  1. Read feedback.csv
  2. Extract user_query column
  3. Create Document objects with metadata
  4. Generate embeddings using sentence-transformers model
  5. Store in FAISS vector index
  6. Save to /faiss_feedback_index/ directory

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Button click doesn't trigger generation
Solution: Check browser console for errors, verify JWT token is valid

Issue: "Error generating embeddings" message appears
Solution: Check if create_embedding.py or feedback_embedding.py exists
         Verify college_policy.txt or feedback.csv exists
         Check file permissions

Issue: "Generating..." text never disappears
Solution: Check server logs for timeout errors
         Refresh page if stuck
         Check if Python script is installed correctly

Issue: Embeddings not improving search results
Solution: Ensure college_policy.txt has good content
         Try regenerating embeddings again
         Check if model is cached correctly in hf_cache/
