## 🚀 How to Use

### For Admins:

**Search Policies:**
```
1. Go to Admin Panel
2. Click "Knowledge Base" tab
3. Enter search term (e.g., "Pearson")
4. Select language (English/Hindi)
5. Click "Search"
6. View results
```

**Add Policy:**
```
1. Scroll to "Add New Policy" section
2. Enter Question (policy title)
3. Enter Answer (detailed information)
4. Select Language
5. Click "Add Policy"
```

**Edit Policy:**
```
1. Search for policy
2. Click "Edit" button
3. Modify text in expanded textarea
4. Click "Save Changes"
```

**Delete Policy:**
```
1. Search for policy
2. Click "Delete" button
3. Confirm in dialog
4. Policy removed
```

### For Students:

Ask the chatbot about:
- Library access (OpenAthens, EBSCO, IEEE, etc.)
- Registration procedures
- Exam information
- Fee payment options
- Course selection
- Grade calculations
- And 135+ more topics!
