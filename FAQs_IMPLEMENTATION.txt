================================================================================
OVERVIEW
================================================================================

The FAQs feature enables:

1. **STUDENTS (ChatUI.html)**
   - View all FAQs in a collapsible accordion format
   - Search FAQs by question or answer text
   - Expand/collapse individual FAQs
   - Access via "FAQs" button in the sidebar

2. **ADMINS (admin.html)**
   - View all FAQs in a formatted list
   - Create new FAQs (Add operation)
   - Edit existing FAQs (Update operation)
   - Delete FAQs (Delete operation)
   - Search FAQs to filter the list
   - Access via "FAQs Management" tab

================================================================================
USER WORKFLOWS
================================================================================

STUDENT WORKFLOW: VIEW FAQs
===========================
1. Student clicks "FAQs" button in sidebar
2. FAQs modal opens
3. FAQs load from backend (GET /faqs)
4. Student can:
   a. Click any question to expand answer
   b. Click again to collapse
   c. Use search box to filter FAQs
   d. Click X or outside modal to close

ADMIN WORKFLOW: MANAGE FAQs
============================
1. Admin clicks "FAQs Management" tab
2. FAQs page loads
3. FAQs fetch from backend (GET /admin/faqs)
4. Admin can:
   a. **Search**: Type in search box to filter
   b. **Add**: Fill question/answer form and click "Add FAQ"
      - PUT /admin/faqs triggered
      - faqs.json updated immediately
   c. **Edit**: Click "Edit" button on any FAQ
      - Browser prompt for new question/answer
      - PUT /admin/faqs/<id> triggered
      - faqs.json updated
   d. **Delete**: Click "Delete" button
      - Confirmation dialog appears
      - DELETE /admin/faqs/<id> triggered
      - faqs.json updated
